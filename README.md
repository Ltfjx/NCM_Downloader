# NCM_Downloader
简易网易云音乐下载工具  
可绕过付费下载歌曲（不支持需要vip才能听的曲子）  

## 下载
https://github.com/Ltfjx/NCM_Downloader/releases/  

## 使用方式  
首先获取网易云歌曲ID  
使用客户端分享功能或网页端获取链接，例如  
https://<span></span>music.163.com/#/song?id=**1308778268**  
那么**1308778268**就是这首歌的ID  
然后打开exe，输入网易云歌曲ID，回车  
这首歌就会被保存到软件当前运行目录下
